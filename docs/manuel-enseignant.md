# 🏫 Manuel Enseignant - Espace Éducation

Ce guide vous aide à gérer vos notes et vos classes via l'application Bull ASUR.

## 1. Accès à vos Matières

En tant qu'enseignant, votre accès est automatiquement restreint aux matières qui vous sont assignées par l'Administrateur.

1. Dès la connexion, vous voyez la liste de vos **Matières actives**.
2. Cliquez sur une matière pour voir le volume horaire, les coefficients et la liste des étudiants inscrits.

## 2. Saisie des Notes

Vous êtes responsable de la saisie des notes pour les matières que vous enseignez.

### Processus de saisie
1. Cliquez sur **Saisir les notes** dans le menu de la matière.
2. Choisissez le type d'évaluation (CC ou EXAM).
3. Entrez les notes. Le système accepte les décimales (ex: 14.5).
4. Cliquez sur **Enregistrer**.

**Note sur les retards** : Si la période de saisie est clôturée par l'administration, vous ne pourrez plus modifier les notes. Contactez le Secrétariat en cas de besoin.

## 3. Suivi et Statistiques

L'application vous permet de visualiser en temps réel la réussite de vos étudiants dans vos matières :
- **Moyenne de classe** : Automatiquement mise à jour.
- **Taux de réussite** : Pourcentage d'étudiants ayant validé la matière (> 10/20).
- **Signalement d'absences** : Vous pouvez enregistrer les absences durant vos cours pour que le Secrétariat puisse les justifier ou appliquer les pénalités.

## 4. Vos Informations

Assurez-vous que vos informations (Email, Matricule) sont correctes. Votre Email est utilisé pour vos identifiants de connexion Supabase.

---
> [!IMPORTANT]
> Vos accès sont protégés par des règles de sécurité strictes (RLS). Vous ne pouvez pas consulter les notes des matières que vous n'enseignez pas, assurant ainsi la confidentialité des données entre collègues.
