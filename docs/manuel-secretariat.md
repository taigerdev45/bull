# 📝 Manuel Secrétariat - Gestion des Notes et Étudiants

Le Secrétariat est le pilier opérationnel de l'application. Ce manuel vous guide à travers les processus de gestion académique.

## 1. Interface et Navigation

L'interface est conçue pour être intuitive. Tous vos outils sont accessibles via la **Barre Latérale (Sidebar)** à gauche de l'écran. 
Le **Tableau de Bord** principal vous offre une vue analytique en temps réel :
- **Effectifs** : Nombre total d'étudiants et d'enseignants.
- **Absences** : Volume horaire des absences enregistrées.
- **PVs** : Suivi des PVs de délibération générés.

## 2. Gestion des Étudiants et Enseignants

### Inscription et Personnel
- **Étudiants** : Utilisez le lien **Étudiants** de la barre latérale pour inscriptions et listes.
- **Enseignants** : Gérez le corps enseignant via l'onglet **Enseignants**. Vous pouvez créer des comptes et assigner des matières spécifiques à chaque personnel.
- **Référentiels** : Accédez à **Référentiels** pour configurer les UEs et les matières. Pour chaque matière, vous pouvez désormais sélectionner un **Enseignant Responsable** dans une liste déroulante pour lui donner les droits de saisie.

## 3. Saisie des Notes (Évaluations)
Le système gère trois types d'évaluations : **CC (Contrôle Continu)**, **EXAMEN**, et **RATTRAPAGE**.

### Saisie via Sidebar
1. Cliquez sur **Saisie Notes** dans la barre latérale.
2. Sélectionnez une **Matière**.
3. Saisissez la note (de 0 à 20) pour chaque étudiant présent.
4. **Calcul Automatique** : Dès que vous enregistrez, le système recalcule automatiquement la moyenne de la matière, de l'UE et du semestre en cascade.

## 4. Gestion des Absences
Accédez à **Absences** dans la barre latérale.
- Enregistrez les heures d'absence hebdomadaires par matière.
- Le système calcule automatiquement les pénalités sur les moyennes.

## 5. Résultats et Bulletins

### Délibération
Via **Délibérations**, consultez le statut de chaque étudiant (Admis, Redoublant, etc.). Les compensations sont appliquées automatiquement selon les règles métier.

### Édition des Bulletins
1. Accédez à **Bulletins** dans la barre latérale.
2. Recherchez un étudiant.
3. Cliquez sur **Générer PDF** pour exporter le document officiel.

---
> [!TIP]
> Si une note de **Rattrapage** est saisie, elle remplace intégralement la moyenne initiale de la matière, quel que soit le score précédent.
